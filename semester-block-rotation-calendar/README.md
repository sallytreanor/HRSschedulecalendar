## Semester Block Rotation Calendar

A simple static web app (no build tooling) for editing a block schedule on an 8-day rotation.

### How to use
1. Open `index.html` in a modern browser.
2. Click any **instructional** date to edit the 4 block periods for that date’s rotation day.
3. Saved values apply to **all dates** that share the same rotation day automatically.

### Data storage
- Uses `localStorage` in your browser.
- Use the **Export** / **Import** buttons to back up or transfer your template.
